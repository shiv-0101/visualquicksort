Randomized Quick Sort

Algorithm:

RandomizedQuickSort(A, low, high)

If low < high:

    Set r = Random(low, high)

    Swap A[r] with A[high]

    Set pivot = A[high]

    Set i = low - 1

    For j = low to high - 1:

        If A[j] ≤ pivot:

            Set i = i + 1

            Swap A[i] with A[j]

    Swap A[i + 1] with A[high]

    Set p = i + 1

    Call RandomizedQuickSort(A, low, p - 1)

    Call RandomizedQuickSort(A, p + 1, high)

Exit

Time Complexity:

Best: O(n log n)

Average: O(n log n)

Worst: O(n²)
